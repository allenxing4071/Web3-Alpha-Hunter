## 文档搜索技巧

### 按主题搜索
```bash
# 搜索所有包含"API"的文档
grep -r "API" docs/

# 搜索数据库相关文档
grep -r "数据库" docs/
```

### 按文件名搜索
```bash
# 查找所有markdown文件
find docs/ -name "*.md"

# 查找包含"快速"的文件
find docs/ -name "*快速*.md"
```

### 按修改时间搜索
```bash
# 最近7天修改的文件
find docs/ -name "*.md" -mtime -7

# 最近修改的5个文件
find docs/ -name "*.md" -type f -print0 | xargs -0 ls -lt | head -5
```
